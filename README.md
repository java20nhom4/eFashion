## Technology

**1. Backend**
  - Java 11
  - Spring Boot 2.7
  - Spring MVC model
  - Spring Data, Hibernate
  - Spring Security

**2. Database**
  - MySQL

**3. Other Technology**
- RESTful API
- JSON Web Tokens
- JavaMail
- jQuery Ajax call API

**4. Tool**
  - Visual Studio Code
  - IntelliJ IDEA 2021.1.3
  - Postman
  - Draw.io for ERD Diagram
